# heyalexchoi.com

Personal website built with Nextra blog theme, deployed to Cloudflare Pages.

## Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

This generates a static export in the `.next` directory.

## Deployment to Cloudflare Pages

1. Connect your Git repository to Cloudflare Pages
2. Set the build command: `npm run build`
3. Set the build output directory: `.next`
4. Deploy!

The site will be available at your configured Cloudflare Pages domain (e.g., heyalexchoi.com).
